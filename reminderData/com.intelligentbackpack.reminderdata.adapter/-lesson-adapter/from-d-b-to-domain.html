fun Lesson.fromDBToDomain(subjects: List<Subject>): EventAdapter.Lesson

Converts a DBLesson to a EventAdapter.Lesson. if the DBLesson is a DBLesson with the same DBLesson.fromDate and DBLesson.toDate then it will be converted to a EventAdapter.DateLessonImpl. if the DBLesson is a DBLesson with different DBLesson.fromDate and DBLesson.toDate then it will be converted to a EventAdapter.WeekLessonImpl.

Return

The converted DBLesson to a EventAdapter.Lesson.

Parameters

subjects

The list of subjects to get the name of the subject from the subject id.
